# Services

Backend services for Sentient v8.

Planned services (high level):

- `sentient-core` real-time scheduler (Rust)
- `sentient-api` shared REST/WebSocket API surface
- `sentient-auth` central RBAC and login
- `sentient-notify` push notifications (FCM)
- `osc-bridge` OSC delivery to SCS with acks/retries
